# reddio
Reddio is a high performance parallel Ethereum-compatible Layer 2  

## Build & Run
### Prerequisites
- go 1.21
### Ethereum compatible
- go-ethereum v1.14.0

### Source code Build & Run
```shell
git clone git@github.com:reddio-com/reddio.git
cd reddio && make build

./reddio
```

### Docker Pull & Run
```shell
docker pull ghcr.io/reddio-com/reddio:latest
docker-compose up
```
